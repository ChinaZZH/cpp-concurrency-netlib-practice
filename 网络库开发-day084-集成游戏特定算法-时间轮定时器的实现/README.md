# Day84： 集成游戏特定算法  时间轮算法来实现定时器

## 核心收获

-- 1.  实现秒级精度的时间轮定时器，每一个时间轮都含有时针，分针，秒针。
 
-- 2.  实现增加定时器 取消定时器的接口。
 
-- 3. 每一帧判断当前的秒数是否发生变化，发生变化了则秒针会发生变化。

-- 4. 可能因为秒针变化还带动了时针和分针变。， 如果时针发生了变化，则将挂在时针下面的定时器任务分发到下面的对应的分针中去。如果分针发生了变化则将挂载的定时器任务分发到秒针下面去。

-- 5. 取出变化后的秒针上挂在的定时器任务去执行。
