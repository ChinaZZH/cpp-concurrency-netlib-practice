# Day88： 集成游戏特定算法  帧同步(下) -确定性模拟

## 核心收获

-- 1.  帧同步分为两部分，一个是输入采集和转发，另一个就是这个确定性模拟。
 
-- 2.  输入采集和转发 带来的是统一的输入(客户端一定频率比如30毫秒发数据给服务端，而服务端每50毫秒采集当前帧索引的数据进行适当的广播)。

-- 3.  确定性模拟带来的是统一的数学运算规则(通过fixed库来统一浮点数的计算结果 通过int64来模拟浮点数来实现)。
 
-- 4.  采用 Q24.40 定点数  也就是int64类型，高24位代表了整数，而低40位代表了浮点数。
