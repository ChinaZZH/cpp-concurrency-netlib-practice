# Day4： 无锁队列 

## 核心收获

-- 无锁队列分成用链表实现和用循环环形数组实现，下面分别用两个文件来分别讨论。
## part_1. 通过链表实现 无锁队列的学习和实现
## 核心收获 
 -- 1. 增加一个虚拟节点，通过增加虚拟节点的方式避免了在pop和push里面判断队列是否为空的判断。

-- 2. 因为避免了pop和push是否判断为空的情况，所以push只需要使用tail不需要使用head.同理pop只需要使用head不需要使用tail.避免了不同函数对共享数据的竞争。

-- 3. exchange函数设置新值并且返回旧值，也是 读-改-写的模式，内存序可以设置为 std:memory_order_release 也可以 std:memory_order_acq_rel.

-- 4. 析构清理所有节点，同时构造的时候要创建虚拟节点同时指向头结点和尾结点。

##代码 -- ListMpcsQueue.cpp

##测试 -- 一切正常。

## part_2. 通过循环环形数组实现的无锁队列
 -- 1. 当前版本是可用的数组版本，目前没发现啥异常， 使用了版本号来实现。
 
 -- 2. 之前尝试过其他版本，问题较多，存在这数据回绕，活锁，等等各种问题，优化到目前版本
 
 -- 3. 目前push的实现方式是先比较版本号是否符合(当前版本号，同时需要pop完成版本号才会+2 新的push才有效)
 
 -- 4. 版本号判断完成就需要马上锁定tail
 
 -- 5. push和pop的时候版本号不满足的话就return false.避免使用自旋锁或者互斥锁来等待来抢占CPU时间和减少并发数量。
 
 -- 6. push或者pop失败的话，就需要外面调用层一直执行直到成功。
 
 -- 7. 不推荐数组这种形态，边界问题太多。
