## 核心收获

-- 1.1. 定义第一条字节码指令并实现Make指令编码器

-- 2.1. 编码器支持中缀表达式

-- 2.2. 通过Dissamble支持反汇编，将字节码翻译成中间语言(汇编语言)

-- 2.3. 支持编译，将语法树的结点node根据不同的语句和表达式转化为汇编语言并且同时格式化为字节码

-- 2.4. 编译完成后，虚拟机运行的时候要从这个编译对应中获取编译完成的字节指令码 和 常量池。

-- 3.1. 支持前缀表达式的的编译和字节码生成

-- 3.2. 支持字面量表达式的的编译和字节码生成
